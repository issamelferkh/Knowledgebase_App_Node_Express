# Knowledgebase_App_Node_Express
This is a knowledgebase app used in the "Node.js & Express From Scratch"

## Technologies
- Node.js
- Express
- Express Messages, Session, Connect Flash & Validation
- MongoDB & Mongoose
- Pug Templating
- Passport.js Authentication
- BCrypt Hashing
